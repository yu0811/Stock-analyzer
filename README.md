# 株式総合分析ツール

銘柄コードを入力すると、中小企業診断士的な視点でのファンダメンタル分析・テクニカル分析・
総合診断コメントを表示するStreamlitアプリです。**課金ゼロ・完全無料でも一通り使えます。**
Claude(Anthropic API)は、より自然な文章の診断コメントが欲しい場合の任意のオプションです。

## 画面構成

- **サイドバー**: お気に入り銘柄一覧（クリックで即切り替え）。銘柄名の下の「☆お気に入りに追加」ボタンで登録・解除
- **ファンダメンタル分析**: PER・PBR・ROE・自己資本比率・流動比率・キャッシュフロー・過去5年の業績推移・来期の会社予想、安全性/収益性/効率性/成長性の5段階評価（★表示＋根拠の説明文）、PPM(プロダクト・ポートフォリオ・マネジメント)観点の簡易事業分類（花形/金のなる木/問題児/負け犬）
- **テクニカル分析**: ローソク足＋移動平均線(5/25/75/200)・ボリンジャーバンド・一目均衡表（切替式）、出来高、MACD、RSI、信用倍率（融資残高÷貸株残高）
- **総合診断**: 上記すべて（5段階評価・PPM分類含む）を組み合わせた**無料のルールベース診断が自動表示**されます。Anthropic APIキーを設定した場合のみ、追加でClaude(AI)による自然文の診断コメントも生成できます（その操作をした場合のみ課金）。

お気に入りは`favorites.json`に、信用倍率の蓄積履歴は`margin_history/`に、それぞれこのフォルダ内にローカル保存されます（`.gitignore`済みなのでGitHubには上がりません）。

## 完全無料で使う場合

**APIキーも登録も一切不要です。** `pip install -r requirements.txt` してそのまま `streamlit run app.py` すれば、株価・テクニカル指標・信用倍率(貸借銘柄のみ)・ルールベースの総合診断まで動きます。財務データ・会社予想だけはJ-Quantsの登録(無料プランでOK)が必要です。Claude APIキーは最後まで一切設定しなくて問題ありません。

## 信用倍率データについて（無料・登録不要）

信用倍率は、日本証券金融(日証金)が**無料・登録不要**で公開している「銘柄別残高一覧」CSV(`zandaka.csv`)から取得します。J-Quantsのように12週遅延したり有料プランが必要になったりすることはありません。

ただしこのCSVは「最新1日分」のスナップショットしか提供していないため、過去の推移グラフは**アプリを使うたびに1日分ずつローカルに記録を蓄積して自前で育てていく**方式にしています（`margin_history/` フォルダに保存されます）。使い始めてすぐは点が1つしかありませんが、日々使ううちに折れ線グラフとして育っていきます。

なお対象は「貸借銘柄」（比較的流動性の高い銘柄）に限られ、貸借銘柄でない銘柄はこのデータの対象外です（アプリ上は「見つかりませんでした」と表示されます）。

最初から数年分の履歴が欲しい、あるいは貸借銘柄以外もカバーしたい場合は、J-Quants Standardプラン（3,300円/月）にすると週次の信用残高データがフル履歴で取得できます。その場合はこちらを優先して自動的に使う設計にしてあります（J-Quantsのトークンを設定していれば優先、なければ日証金の無料データにフォールバック）。

## 必要なもの（事前準備）

**両方とも任意です。何も登録しなくてもアプリ自体は動きます**(株価・テクニカル・信用倍率・ルールベース診断のみ利用可)。J-Quantsは財務データ・会社予想を使いたい場合に、Anthropic APIキーはAI(Claude)による自然文の診断コメントも追加したい場合に、それぞれ登録してください。

### 1. J-Quants（任意）— 財務データ・会社予想・信用残高の全履歴

1. https://jpx-jquants.com/ でメールアドレス登録
2. マイページから **refresh token** を発行してコピー
3. プランは目的に応じて選択:

   | プラン | 月額 | 財務データ・会社予想 | 信用取引週末残高（全履歴） |
   | --- | --- | --- | --- |
   | Free | 無料 | 取得可（12週遅延） | 使えない（アプリは自動的に日証金の無料データにフォールバック） |
   | Light | 1,650円 | 取得可（遅延なし） | 使えない |
   | Standard | 3,300円 | 取得可（遅延なし） | 使える（遅延なし・全履歴）＋業種別空売り比率も追加 |

   信用倍率だけが目的なら、上記の「信用倍率データについて」の通り無料の日証金データで足りるため、**J-Quantsのアカウント登録自体をスキップしてもアプリは動きます**（ファンダメンタルタブの財務データ・会社予想が空になるだけです）。

### 2. Anthropic API（Claude）— 任意。AIによる自然文の診断コメントを追加したい場合のみ

**設定しなくてもアプリは問題なく使えます。** 設定しない場合、総合診断タブにはルールベース(無料)の診断コメントだけが表示されます。それで十分という場合はこの手順はスキップしてください。

AIによる診断コメントも欲しい場合のみ:
1. https://console.anthropic.com/ でアカウント作成
2. APIキーを発行（従量課金。銘柄を調べるたびに数円程度のコストが発生します）
3. 既定モデルは `claude-sonnet-5`（速度とコストのバランスが良いため）。`ai_commentary.py` の `DEFAULT_MODEL` を変更すれば `claude-opus-4-8` 等に切り替え可能です。

### 3. 株価データ（yfinance）— 追加登録不要

無料・登録不要で使えます。ただし Yahoo Finance 側の仕様変更で稀に取得できなくなることがあります。

## ローカルで動かす

```bash
cd stock_analyzer
pip install -r requirements.txt
cp .streamlit/secrets.toml.example .streamlit/secrets.toml
# secrets.toml を開いて、上記で取得したトークン・APIキーを貼り付ける
streamlit run app.py
```

ブラウザが自動的に開き、`http://localhost:8501` でアプリが起動します。
毎回コマンドを打つのが面倒な場合は、上記2行をまとめた `.command`（Mac）や `.bat`（Windows）ファイルを作ってダブルクリック起動にすることもできます。

## スマホからも使えるように公開する（Streamlit Community Cloud）

1. このフォルダの中身をGitHubリポジトリにpushする（**secrets.tomlは絶対に含めない** — `.gitignore`済み）
2. https://share.streamlit.io/ にアクセスし、GitHubアカウントで連携
3. 対象リポジトリ・ブランチ・`app.py` を指定してデプロイ
4. デプロイ後、アプリの **Settings > Secrets** に `.streamlit/secrets.toml.example` と同じ形式でトークン・APIキーを貼り付ける
5. 発行されたURLをスマホのホーム画面に追加すれば、アプリのように起動できます

## 注意点・既知の制約

- **信用倍率は無料の日証金データで取得できますが、対象は貸借銘柄のみ・履歴はアプリ使用開始日からのみです。** 全銘柄・全履歴が欲しい場合はJ-Quants Standardプラン（3,300円/月）が必要です。
- **`data_sources.py` の `FIELD_MAP`（J-Quants `/fins/statements` のレスポンス項目名）は公式ドキュメントに基づく想定値です。** この開発環境ではネットワーク制限により実際のAPIレスポンスに対する実地検証ができませんでした。初回セットアップ後、`debug_dump_raw_statement()` 関数で実際のJSONを1度出力し、項目名が一致しているか確認してください。ずれがあれば `FIELD_MAP` の値を実際のキー名に書き換えてください。
- **同様に `data_sources.py` の `TAISYAKU_COL_*`（日証金CSVの列位置）も、解説ページの記述とサンプル行の手計算による推定値です。** この開発環境ではネットワーク制限により実際のCSVの内容を完全には検証できませんでした。初回実行後、`debug_dump_taisyaku_raw()` 関数で実際の行データを1度確認し、列位置がずれていないか確認してください。
- **空売り比率は銘柄別の日次データとして無料で公開されていないため、今回は実装を見送りました。** JPXが公表する空売り集計は市場全体の日次PDFのみで、個別銘柄の空売り比率は含まれません。
- 本ツールは情報提供のみを目的としており、投資助言ではありません。表示される診断コメントは断定的な売買判断を含まないよう設計していますが、最終的な投資判断はご自身の責任で行ってください。
- 公開URLにする場合、`APP_PASSWORD` をsecretsに設定すると簡易的なパスワード保護がかかります（未設定なら誰でもアクセス可能な状態になります）。

## ファイル構成

```
stock_analyzer/
├── app.py                       # Streamlit本体（3タブUI＋お気に入りサイドバー）
├── data_sources.py               # yfinance / J-Quants / 日証金 からのデータ取得
├── indicators.py                 # テクニカル指標の計算（MA/MACD/RSI/BB/一目均衡表/信用倍率）
├── fundamentals.py                # 財務比率・5段階評価・PPM分類の計算
├── free_diagnosis.py              # 無料・ルールベースの総合診断コメント生成（課金なし）
├── ai_commentary.py               # Claude APIによる診断コメント生成（任意・課金あり）
├── favorites.py                   # お気に入り銘柄の保存・読み込み
├── requirements.txt
├── .streamlit/secrets.toml.example
└── README.md
```
